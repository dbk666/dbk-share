# dbk-share
盘点可自由上传分享影视作品/音乐/电子书/影评书评/软件/游戏的网站。(长期更新)

# 前言：
不以**自由与发展为前提的道德**毫无意义，原po的初衷是为对抗审查尽一份薄力，顺便为同时手头拮据但仍有精神需求的朋友指条有争议的路，
若您回过头来条件允许，请考虑**为正版补齐付费。**

建议勿传播**非汉语或非汉化作品（尤其是照搬主流民主国家之生肉）**，拨云见日前，大家要同饮食，勿要砸锅。

# **影视类**

### **🇷🇺[VK video](https://vkvideo.ru/)**
```
建于2006 ,俄罗斯最大社交平台VK的视频站，
创始人极端奉行自由主义和无版权主义（虽已离任，但精神永存）
加之俄民间和法律悠久独特的版权观，搬运已蔚然成风二十载，尾大不掉。
值得一提的是，VK只支持手机号注册
```

### **🇺🇸[Odysee](https://odysee.com/)**
```
建于2020，去中心化上传站.
得益于去中心化 极难被和谐, 
故odysee可作为其余三家都无法上传的极少数作品之备选去处(仅建议上传汉语或汉化作品)
```


### **🇼🇸[伊莉影片](https://www.eyny.com/video)**
```
建于2003，台湾eyny论坛的子站，汉语世界里大名鼎鼎的海盗视频站，
某些国家IP访问不畅（如日本看不了情色以外的板块，美日法不登录无法预览）
值得一提的是：eyny目前接受的注册邮箱的范围较窄,
只有主流的几家: 腾讯/网易,谷歌/苹果/微软/雅虎/AOL.

另外，根据eyny的政策：
禁止上传范围参见https://www.eyny.com/video.php?mod=disclaimer
清晰度随播放量解锁（默认240p，100播放360p，1000播放720p）
视频年播放＜100会被删片
```


### **🇨🇳[爱壹帆](https://www.yfsp.tv/)**
```
建于2013，汉语世界里大名鼎鼎的海盗视频站.
（中国内地访问需将梯子开全局代理,否则无法访问）
值得一提的是，爱壹帆虽明面宣称可上传视频，但有较高的未知门槛。
```

# **电子书类**

### **🇺🇳[Z-Library](https://mastodon.social/@Z_Lib_official)**
```
建于2009，著名的 起源于俄的电子书网站。因域名常换，故贴Mastodon官方主页。
```

### **🇺🇳[The Library Genesis](https://social.datalabour.com/@libgenbot)**
```
建于2008, 前身是苏联时期的地下印刷组织，享誉世界的影子图书馆项目.
和zlib一样，域名常换，故贴Mastodon发布页.
```

### **🇺🇳[Anna’s Archive](https://annas-archive.org/)**
```
建于2022，一家自由、非营利的影子图书馆元搜索引擎，因22年FBI查封ZLib而创立。
值得一提的是，安娜档案涵盖了大部分前文libgen和zlib的资源，
但其暂时只支持大规模、不被其他网站接受的内容，否则仍需上传zlib或libgen。
```

# **作品评价/撰写类**

### **🇺🇳 [Neodb](https://neodb.social/)**
```
建于2021，墙外汉语世界的“满血版豆瓣”，内容全面且表达自由，并支持自建缺失的作品条目信息。
NEODB开源免费，靠捐赠与情怀运营，如体验不错，大飙客建议长期赞助一点小钱
NEODB受捐链接：https://ko-fi.com/neodb
```

### **🇺🇳 [Matters](https://matters.town/)**
```
建于2018，墙外汉语世界去中心化的长文撰写平台，逼格与质量都很高，可以当做Neodb无长文功能之补充。
Matters与加密货币合作，读者可通过拍手按钮让作者获得LikeCoin，
亦或直接为作者贡献指定数额的金钱支持。
```



# **软件&游戏类**

- **tips：虽盘点网站皆经过大飙客之遴选，但因”学习版”软件或游戏无可避免有安全性良莠不齐之嫌，请在一定心理预期与安全意识之前提下查看以下内容。**

### **🇨🇳[小叽资源](https://steamzg.com/)**
```
建站时间不可考，整个汉语世界最棒的软件/游戏共享站(服务器位于墙外)
因服务器成本激增，实行定期发放注册资格。
因安全性考虑，暂停了新帐户的独立投稿权限，改为发给站长审核。
```

### **🇺🇳[Filecr](https://filecr.com/)**
```
建于2008，Filecr算是软件/游戏站较专业化的一类，致力于提供一个全面的平台。
虽然未开放注册，但允许提交持有的软件/游戏(站方先审后发)，
或发起软件收录请求，他们似乎很乐于处理请求。
注：Filecr的内容均需以英文形式收录；域名已被部分国家墙掉，可能需要vpn。
```

### **🇷🇺[Rutracker](https://rutracker.org/)**
```
建于2004，俄罗斯最屌的资源分享站。
2015年11月被莫斯科法院查封，俄乌战争期间解封。
rutracker界面粗旷，资源多以bt种子之形式发布，毛味十足。
可能是上传自由度最高的资源共享类大平台，允许用户自主发布。
```
